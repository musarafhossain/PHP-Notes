# PHP Supports various Databases

- **MySQL**
- MS-SQL
- SQLite
- MongoDB
- Oracle OCI8
- **PostgreSQL**

There are three main API options when considering connecting to a MySQL database server:
- PHP's MySQL Extension (Depricted)
- PHP's mysqli Extension
- PHP Data Objects (PDO)
